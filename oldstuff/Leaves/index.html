Controls:

Use the Up, Down, Left, Right arrows to select an action and then use the mouse to click and apply that action.

 

Up: Will allow you to change the rotation of a leaf when clicked

Down: Will allow you to drag a leaf when clicked

Left: Will allow you to change the leaf within the group when clicked

Right: Will allow you to change the leaf group when clicked.
